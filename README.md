# ⚡ EcoCharge - EV Battery Degradation Prediction

EcoCharge is a machine learning powered web application that predicts **Electric Vehicle battery degradation** based on operational and environmental factors.

The project helps users estimate battery health and understand how different parameters affect long-term battery performance.

## 🧠 Machine Learning

The model predicts degradation using parameters such as:

- Charging cycles
- Temperature
- Depth of discharge
- Charging rate
- Battery age

Algorithms used may include:

- Random Forest
- Gradient Boosting
- XGBoost (if used)
